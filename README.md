# 📘 Multiplication Table Generator (Java)

## 🚀 Project Overview
This is a simple Java console application that generates a multiplication table for a number entered by the user.  
The table is printed from **1 to 20** using a `for` loop.

---

## 🛠️ Features
- Takes user input using `Scanner`
- Generates multiplication table
- Displays results in a clean format
- Uses basic Java concepts (loops, input, arithmetic)

---

## 💻 How It Works
1. User enters a number
2. Program reads input
3. `for` loop runs from 1 to 20
4. Each iteration prints multiplication result

---

## 📌 Example Output

Enter a number to print its multiplication table:
7
7 x 1 = 7
7 x 2 = 14
7 x 3 = 21
...
7 x 20 = 140
